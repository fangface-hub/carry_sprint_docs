# carry_sprint_docs

Specification and design documents for CarrySprint.

## Purpose

This repository defines the specifications and architecture for CarrySprint.  
The implementation is generated by AI based on these specifications.  
This repository contains only documents that define the system.

## Principles

- Structure first
- One sentence per statement
- No ambiguity
- No hidden state
- No branching in explanations
- Reproducible environment
- AI-driven implementation
- Separation of documents and source code

- **CLI client** — command-line interface that communicates with the local server

## Scope

This repository does not contain source code.
